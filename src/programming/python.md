# Python

## Basic Syntax Examples 

### Print
```python
print("Hello, World!")
```

### Lists
```python
my_list = [1, 2, 3, 4, 5]
print(my_list)
print(len(my_list))

my_list.append(6)
print(my_list)
```
    - Lists are mutable
    - Lists are ordered
    - Lists can contain duplicates
    - Lists can be nested
    - Lists are dynamic

### Tuples
```python
my_tuple = (1, 2, 3, 4, 5)
print(my_tuple)
```
