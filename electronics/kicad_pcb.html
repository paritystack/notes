KiCad PCB

Overview

KiCadвҖҷs PCB editor (pcbnew) is where the logical schematic becomes a physical board. You import the netlist from the schematic, assign positions to footprints, draw the board outline, route copper traces, fill a ground plane, run a Design Rules Check (DRC), and export Gerber files for fabrication. This page follows the USB-powered LED driver from KiCad Schematic through to ordering from JLCPCB. See Circuit Design for the underlying PCB concepts.

Workflow

1. Open the PCB Editor

From the KiCad Project Manager, click the PCB editor icon, or open project.kicad_pcb directly.

2. Import Schematic (Update PCB from Schematic)

Tools вҶ’ Update PCB from Schematic (F8). KiCad places all footprints in a cluster off-board with ratsnest lines showing unrouted connections.

  After import вҖ” ratsnest view:

If a footprint is missing, the error message names the component вҖ” go back to the schematic and assign the footprint, then re-run F8.

3. Draw Board Outline

Select layer Edge.Cuts in the layer dropdown (right side panel or the layer selector at top).

Place вҶ’ Rectangle (or press R then draw), or Place вҶ’ Line for irregular shapes. Draw a closed shape вҖ” the fab router cuts exactly along this line.

For the example: Draw a 30 Г— 20 mm rectangle. Verify it is closed (zoom in on corners).

4. Place Components

Drag footprints inside the board outline. Press G to move while keeping ratsnest connected, R to rotate.

Placement strategy:

  • Keep components close to the connections they serve вҖ” short ratsnest lines mean short traces
  • Orient connectors at board edges (J1 at the left edge so the USB plug overhangs)
  • Group power components together (not relevant here, but important in larger designs)
  • Verify component orientation: LED cathode (shorter leg) toward GND; check the footprint silkscreen for polarity markers

5. Route Traces

Press X to start the interactive router. Click the start pad, route, click the end pad.

  • Select the layer in the toolbar before routing (F.Cu for this example)
  • Press / to switch layers mid-route (inserts a via automatically)
  • Press W during routing to change trace width
  • Press Esc to abandon an in-progress trace

Route the LED driver in this order:

  1. VBUS net:   J1.VBUS в”Җв”Җв”Җв”Җ R1.pad1   (F.Cu, 0.5 mm вҖ” carries USB 5V)
  2. LED_A net:  R1.pad2 в”Җв”Җв”Җв”Җ D1.anode  (F.Cu, 0.2 mm вҖ” signal current, ~9 mA)
  3. GND net:    J1.GND  в”Җв”Җв”Җв”Җ D1.cathode (F.Cu, or leave for ground plane)

Ratsnest lines disappear as each net is routed. The board is fully routed when no dashed lines remain.

6. Add Copper Fill (Ground Plane)

A copper fill on B.Cu connected to GND acts as a ground plane вҖ” it automatically connects every GND pad and reduces return-path impedance.

Place вҶ’ Zone (Ctrl+Shift+Z):

  1. Select layer B.Cu
  2. Select net GND
  3. Click around the entire board outline to define the fill area
  4. Press B to fill all zones

7. Add Silkscreen Labels

On layer F.SilkS, add text for board identification (name, version, date). The component reference designators (R1, D1, J1) are already placed by the footprint вҖ” move them to avoid overlapping pads or each other.

Verify silkscreen does not overlap any pad (DRC will warn if it does).

8. Run DRC

Inspect вҶ’ Design Rules Checker вҶ’ Run DRC.

Fix all errors before exporting. Common DRC errors:

9. Export Gerber Files

File вҶ’ Fabrication Outputs вҶ’ Gerbers.

Settings for JLCPCB (2025):

  • Layers: F.Cu, B.Cu, F.SilkS, B.SilkS, F.Mask, B.Mask, Edge.Cuts
  • Format: Gerber X2 (or RS-274X)
  • Precision: 6 decimal places
  • Enable вҖңUse Protel filename extensionsвҖқ for JLCPCB compatibility

Also export the drill file: Fabrication Outputs вҶ’ Drill Files вҶ’ Generate Drill File (Excellon format, metric).

  project-B_Cu.gbl       вҶҗ back copper

Zip all files into one archive.

10. Order from JLCPCB

  1. Go to jlcpcb.com вҶ’ Quote Now
  2. Upload the zip file вҖ” the Gerber viewer shows a 3D preview; verify it looks correct
  3. Standard options for a basic 2-layer board:
    • Layers: 2
    • Base material: FR4
    • Thickness: 1.6 mm
    • Surface finish: HASL (cheapest) or ENIG (flatter pads, better for fine-pitch SMD)
    • Copper weight: 1 oz
    • Min track/spacing: 0.2 mm/0.2 mm
  4. Add to cart; minimum order is 5 boards; typical cost $2вҖ“5 for a small board + shipping
  5. Lead time: ~24 h production + 5вҖ“7 day shipping (DHL option available for ~1 week total)

Pitfalls

  • Re-filling zones after edits вҖ” copper fills are not live. Press B again after any routing change; otherwise DRC checks stale copper.
  • Not verifying Gerbers вҖ” always open the exported Gerbers in KiCadвҖҷs Gerber viewer (or gerbv) before ordering. A missing layer is a wasted order.
  • Polarity-sensitive components вҖ” D1 and electrolytic capacitors must match the footprint polarity markings. Check the silkscreen and footprint documentation.
  • SMD on both sides without reflow oven вҖ” if youвҖҷre hand-soldering, keep all SMD on F.Cu. SMD on B.Cu requires two reflow passes or careful hand-soldering while holding the top-side components.
  • Ignoring thermal relief вҖ” a pad inside a ground plane without thermal relief is hard to solder by hand (the plane wicks heat away). KiCad adds thermal reliefs by default; do not disable them for hand-soldered boards.
  • Ordering before reviewing Gerbers вҖ” the board house will make exactly what you send. A missing Edge.Cuts means no board; a mirrored silkscreen means unreadable labels. Review first.
